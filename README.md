## Features (planned)
- 
- Graph data model (nodes, edges, properties)
- Query language (SQL/GraphQL inspired)
- Traversal and pattern matching
- Indexing and persistence

## Installation

```bash
git clone https://github.com/Palladium02/minerva.git
cd minerva
cargo build
```

## Usage

```bash
cargo run
```
